The Problem

The pervasive issue of misplaced shopping trolleys creates a cascade of negative effects for retailers and the public. This persistent problem is multifaceted, impacting businesses financially, posing risks to public safety, and causing daily frustration for shoppers.

The Journey: From Ideation to Solution

lightbulbIdeation & Wireframing

To address the lack of incentive for returning trolleys and the navigational challenge of making return stations more visible, a mid-fidelity prototype of the "Smart Station" was developed. This early design included:

  • Main Display
  • User Interactive Tablet
  • Trolley Sensing Frame

task_altUsability Testing

User testing validated the design and revealed key insights:

üîí User Privacy:

Vouchers displayed on the main public screen raised concerns of privacy.

üñºÔ∏è Aesthetics:

The flickering display was visually jarring and created unnecessary cognitive load.

‚è© Flow Efficiency:

A minigame in the process was found to be a potential bottleneck, slowing down the user journey.

model_trainingKey Iteration

User feedback was used to iterate on the initial concept, shaping the following high-fidelity prototype.

The QR code was relocated to the user interactive tablet to address privacy concerns.

The jarring flickering animation was replaced with a smooth, rolling gradient for an enhanced visual experience.

The minigame was removed and a timer was implemented to streamline the flow and prevent system abuse.
